  • ImageryImageTransition

    Transition effect which will be used when an image downloaded and set by UIImageView extension API in Imagery. You can assign an enum value with transition duration as an item in ImageryOptionsInfo to enable the animation transition.

    Apple’s UIViewAnimationOptions is used under the hood. For custom transition, you should specified your own transition options, animations and comletion handler as well.

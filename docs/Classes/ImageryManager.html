ImageryManager

public class ImageryManager

Main manager class of Imagery. It connects Imagery downloader and cache. You can use this class to retrieve an image via a specified URL from web or cache.

  • defaultOptions

    Default options used by the manager. This option will be used in Imagery manager related methods, including all image view and button extension methods. You can also passing the options per image by sending an options parameter to Imagery’s APIs, the per image option will overwrite the default ones if exist.

    Note

    This option will not be applied to independent using of ImageDownloader or ImageryCache.

    Declaration

    Swift

  • retrieveImage(with:options:progressBlock:completionHandler:)

    Get an image with resource. If ImageryOptions.None is used as options, Imagery will seek the image in memory and disk first. If not found, it will download the image at resource.downloadURL and cache it with resource.cacheKey. These default behaviors could be adjusted by passing different options. See ImageryOptions for more.

    Declaration

    public func retrieveImage(with resource: ImageryResource,
        options: ImageryOptionsInfo?,
        progressBlock: ImageryDownloadProgressBlock?,
        completionHandler: ImageryCompletionHandler?) -> ImageryRetrieveImageTask

    Parameters

    resource

    ImageryResource object contains information such as cacheKey and downloadURL.
    options

    A dictionary could control some behaviors. See ImageryOptionsInfo for more.
    progressBlock

    Called every time downloaded data changed. This could be used as a progress UI.
    completionHandler

    Called when the whole retrieving process finished.

    Return Value

    A ImageryRetrieveImageTask task object. You can use this object to cancel the task.

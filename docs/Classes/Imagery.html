  • setImage(with:placeholder:options:progressBlock:completionHandler:)

    Set an image with a resource, a placeholder image, options, progress handler and completion handler.

    Note

    Both the progressBlock and completionHandler will be invoked in main thread. The CallbackDispatchQueue specified in optionsInfo will not be used in callbacks of this method.

    If resource is nil, the placeholder image will be set and completionHandler will be called with both error and image being nil.

    Declaration

    Swift

    public func setImage(with resource: ImageryResource?,
                         placeholder: ImageType? = nil,
                         options: ImageryOptionsInfo? = nil,
                         progressBlock: ImageryDownloadProgressBlock? = nil,
                         completionHandler: ImageryCompletionHandler? = nil) -> ImageryRetrieveImageTask

    Parameters

    resource

    ImageryResource object contains information such as cacheKey and downloadURL.
    placeholder

    A placeholder image when retrieving the image at URL.
    options

    A dictionary could control some behaviors. See ImageryOptionsInfo for more.
    progressBlock

    Called when the image downloading progress gets updated.
    completionHandler

    Called when the image retrieved and set.

    Return Value

    A task represents the retrieving process.
